AWS Amplify Environment Variables to Set:

ENVIRONMENT = production
SECRET_KEY = super-secure-secret-key-change-this-in-production-$(date +%s)
DATABASE_URL = sqlite:///./delivery_platform_prod.db

Optional (for RDS later):
DB_HOST = your-rds-endpoint.amazonaws.com
DB_PORT = 5432
DB_NAME = delivery_platform
DB_USER = your-db-username
DB_PASSWORD = your-db-password

Instructions:
1. In Amplify Console, go to "Environment variables"
2. Add each variable above
3. Save and redeploy
